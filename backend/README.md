# webpack backend typescript template

* to **install** run:
    ```
    npm install
    ```

* Run in **separate terminal** *respectively*:
    1. to build in **webpack development mode** run with **-w flag**:
        ```
        npm run dev
        ```
    2. to start the **dist/server.js** file
        ```
        npm run server
        ```

* to build in **webpack production mode** run:
    ```
    npm run build
    ```
